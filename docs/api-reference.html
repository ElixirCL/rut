    modules

     Modules

    ElixirCLRut

    A module that helps with validation and formatting of chilean RUT/RUN identifiers. Made by the Elixir Chile Community.

    ElixirCLRut.CheckDigit

    The Check Digit allows to verify if a RUT is correctly formed.

    ElixirCLRut.Formatter

    Cleans and give format to RUT strings.

    ElixirCLRut.Struct

    A struct that holds information of the rut.

    ElixirCLRut.Token

    Token is a struct that is passed between validations. It enables an standarized way of passing down the information.

    ElixirCLRut.Validations

    Stores different validation functions

    ElixirCLRut.Validator

    Provides simple validation helpers.
